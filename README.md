📺 PineStation99 | PTV

An immersive, retro-style CRT TV simulator built with HTML5, CSS, and JavaScript. This project features a curated broadcast of the Metal Tiger X and The Long Road series, complete with authentic Pinec Multimedia station bumps.

🌟 Features

Authentic CRT Look: Includes scanlines, phosphor glow, and a curved screen bezel.

Interactive UI: A clickable "Power On" button with a screen flash and static effect.

On-Screen Display (OSD): Retro green text overlays showing current "tape" titles.

Dynamic Static: Realistic canvas-based noise during transitions or "signal loss."

Animated Environment: A cozy desk surface with ambient lighting that reacts to the screen.

🎬 Featured Content

Metal Tiger X: Jaguar Protocol (Episodes 1-7)

The Long Road (Episodes 1-2)

Pinec Multimedia Station IDs and Bumps

🚀 How to Run

Download the index.html file.

Open it in any modern web browser.

Click the "TURN ON TV" button to start the broadcast.
